Task 1 - Tidy the data

Write up

First, I manually identified the most granular nested item from the f1 object: races. To correctly format the nested structure I converted races to a tibble and leveraged various dplyr functions making use of hoist to cut down on unnecessary unnest_wider calls. The top 10 rows were displayed using head.

Code

val = f1$MRData$RaceTable$Races
# convert to tibble
val = tibble::tibble(val)
# dplyr pipeline start
val = val %>% hoist(
  val,
  race_name = "raceName",
  round = "round",
  date = "date",
  results = "Results"
) %>% # elongate results for each race at driver level
  unnest_longer(results) %>%
  # hoist again to extract final table 
  hoist(
    results,
    givenName = c("Driver", "givenName"),
    familyName = c("Driver", "familyName"),
    name = c("Constructor", "name"),
    position = "positionText",
    points = "points"
  ) %>% # concatenate driver name
  mutate(driver = paste(givenName, familyName, sep = " "),
         position = case_when(
          position == 'R' ~ as.character(NA), #"Ret",
          position == 'D' ~ as.character(NA), #"DQ",
          position == 'E' ~ as.character(NA), #"Excluded",
          position == 'W' ~ as.character(NA), #"WD",
          position == 'F' ~ as.character(NA), #"FTQ",
          position == 'N' ~ as.character(NA), #"NC",
          TRUE ~ position
         )) %>%
  select(race_name,
         round,
         date,
         driver,
         name,
         position,
         points) %>%
  rename(constructor = name)

# format and print table
val %>% head(., 10) %>%

In order to aggregate the correct metric one column, points, was converted to numeric type. A temporary data structure called total_points was created to capture total points by driver over the season.

Race position was determined next. The base table from task 1 was pivoted on race_name to reflect the drivers position in each race. This race table was joined back to the total_points to create the final task objective.

An additional package, kableExtra, was used to format tables using CSS bootstrap styling.

Task 3 - Cumulative Constructors

Write up

First a temporary data frame was created to assign the final season team rankings which will be joined back to final result set.

Team points were calculated for each race of the season based on driver affiliation. After team points, a new column was created to store the cumulative sum of points throughout the season. This was joined back to the initial temp data frame to assign team rankings and final result set sorted based on this ranking.

Code

# temporary dataframe storing final season team rankings
season_rank = val %>% 
  group_by(constructor) %>%
  summarise(season_point_total = sum(points)) %>%
  mutate(ranking = rank(desc(season_point_total)))

# first compute team points for each race (sum of points for each driver on each
# team).
task3_tbl = val %>% ungroup() %>%
  group_by(race_name, date, constructor) %>%
  summarise(season_points = sum(points)) %>%
  arrange(date) %>% ungroup() %>% group_by(constructor) %>%
  inner_join(season_rank, by = "constructor") %>%
  
  # based on the total team points for each race, compute cumulative sum
  mutate(cum_points = cumsum(season_points)) %>%
  select(-date,-season_points) %>%
  pivot_wider(names_from = race_name, values_from = cum_points) %>%
  arrange(ranking) %>%
  select(-season_point_total,-ranking)

# set col names to match wikipedia source
colnames(task3_tbl)[1:ncol(task3_tbl)] <-
  c(
    'Constructor Name',

Task 4 - Visualization

Write up

F1 is a truly unique sport combining the marvels of modern engineering with some of the most gifted reflex-based athletes in the world. Many people might not be aware of the resource disparities which exist between teams who compete in the same races for the same championship crown. Just how unequal are these teams in terms of the seasonal F1 points system awarded based on final race position?

This visualization hopes to display how F1 is dominated by the top team(s) by displaying the performance of drivers and teams in the 2020 F1 season.

Two charts were created for this purpose:

Spiral Team Chart (A):
Team performance through each race is displayed using a circular bar plot. Race abbreviations, denoted along the outside edge, start from 1:00 and move clockwise around the circle in order of race occurrence. Points for each driver are aggregated by team, cumulatively summed over the season, and shown by color. This chart clearly shows the dominance by team Mercedes over the season as the color largest block in each race is affiliated with this team. We see this percentage remain relatively constant from the first few races of the season all the way to the finale.

Spiral Driver Chart (B):
Point disparities amongst drivers and teams are most magnified at season end. To show this lopsided distribution a spiral bar chart was created. Drivers are grouped by constructor and colored according to their affiliation. From this chart one can observe just how dominant Lewis Hamilton (and team Mercedes) really was. Hamilton has almost twice as many points as the next closest opposing driver. This chart also shows the large number of drivers with 0 points, evident by the missing axis tick marks denoting point totals.

Code

############### plot1 - team performance by race (spiral bar) ###############
season = val %>% ungroup() %>%
  group_by(race_name, date, constructor) %>%
  summarise(season_points = sum(points)) %>%
  arrange(date) %>% ungroup() %>% group_by(constructor) %>%
  inner_join(season_rank, by = "constructor") %>%
  mutate(cum_points = cumsum(season_points)) %>%
  select(-season_point_total)

# assign integer denoting order race occurred during season
race_int = val %>% distinct(race_name, date) %>% 
  mutate(race_num = rank(date))%>% select(-date)
team_cir = season %>% inner_join(race_int, by = "race_name") %>% 
  arrange(race_num, cum_points)
